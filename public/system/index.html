BMEG System

The BioMedical Evidence Graph integrates various biological data source into a single property graph to enable easy analysis. The BMEG is powered by the GRaph Integration Platform GRIP. The current installation covers data from TCGA, CCLE, CTDD, GTEX, Ensembl, PathwayCommons and contains almost 32 million vertices.
